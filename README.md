# AIPhotoKey

A macOS application that helps you organize and search through your photo collection using AI-powered features.

## Features
- Scan and index photos from your local directories
- AI-powered photo analysis and categorization
- Real-time scanning progress and control
- Configurable confidence threshold for AI analysis
- Modern SwiftUI interface

## Requirements
- macOS
- Xcode

## Development
This project uses XcodeGen to generate the Xcode project file. Make sure you have XcodeGen installed before building the project.
